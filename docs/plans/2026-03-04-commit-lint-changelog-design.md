# Commit Linting & Changelog Generation

## Problem

Terrarium uses informal Conventional Commits (`feat:`, `fix:`, `chore:`) but has no enforcement. Release notes are auto-generated by GitHub's API, which produces generic output. There is no `CHANGELOG.md`.

## Goals

1. Enforce Conventional Commits via a local git hook
2. Generate a `CHANGELOG.md` from commit history using git-cliff
3. Replace GitHub-generated release notes with git-cliff output in the publish workflow

## Commit Linting

**Tools:** `@commitlint/cli`, `@commitlint/config-conventional`, `husky`

- `commitlint.config.js` extends `@commitlint/config-conventional`
- Allowed types: `feat`, `fix`, `chore`, `docs`, `style`, `refactor`, `perf`, `test`, `build`, `ci`
- `husky` installs a `.husky/commit-msg` hook running `npx commitlint --edit $1`
- `pnpm prepare` script runs `husky` for automatic hook setup on install

## Changelog Generation

**Tool:** git-cliff

**Configuration (`cliff.toml`):**
- Groups commits by type: Features (`feat`), Bug Fixes (`fix`), Documentation (`docs`), Performance (`perf`), Refactoring (`refactor`)
- Hides internal types: `chore`, `style`, `ci`, `build`, `test`
- Strips PR number suffixes, links to GitHub
- Tag pattern: `v*`

**Output:**
- `CHANGELOG.md` at repo root
- `pnpm changelog` script for local generation

## Publish Workflow Changes

Replace the `release-notes` job (lines 145-163 of `publish.yml`) with a git-cliff step that:
1. Installs git-cliff via `orhun/git-cliff-action`
2. Runs `git cliff --latest --strip header` to generate notes for the new tag
3. Patches the GitHub release body with the output

The `publish` and `update-homebrew` jobs remain unchanged.

## Files Changed

- `package.json` — add devDependencies, `prepare` and `changelog` scripts
- `commitlint.config.js` — new, commitlint configuration
- `cliff.toml` — new, git-cliff configuration
- `.husky/commit-msg` — new, commit-msg hook
- `.github/workflows/publish.yml` — replace `release-notes` job
- `CHANGELOG.md` — new, generated from existing commit history
